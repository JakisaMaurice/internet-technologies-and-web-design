we have worked on the following projects

home automation

This project showcases a smart home automation system designed to enhance comfort, security, and energy efficiency. Using microcontrollers like the ESP8266, the system integrates various sensors and IoT-enabled devices to remotely control and monitor lights, temperature, and security features through a smartphone app or web interface. Key features include automated lighting, real-time temperature adjustments, and security alerts, all connected through a Wi-Fi network. This project demonstrates skills in IoT programming, embedded systems, and practical knowledge of smart home solutions.


fire safety system

This project presents a fire safety system aimed at early fire detection and swift emergency response. Utilizing sensors to detect smoke, temperature, and gas levels, the system continuously monitors indoor environments for signs of fire hazards. In case of detection, it sends real-time alerts via SMS or app notifications, activates alarms, and can trigger safety protocols, such as shutting off electrical systems. Built with IoT components and microcontrollers, this project highlights expertise in sensor integration, real-time monitoring, and safety automation.

solar tracking system

This project involves designing a solar tracking system that optimizes solar panel alignment to maximize energy capture throughout the day. Using sensors to detect sunlight intensity, the system automatically adjusts the panel's angle, following the sun's path to increase efficiency compared to fixed panels. Built with microcontrollers and motorized mechanisms, this project demonstrates skills in renewable energy solutions, embedded systems, and efficient power generation, contributing to sustainable energy innovation.
